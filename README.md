# Public Health Surveillance Pipeline (CDC → AWS → Dashboard)

## Project Overview
This project builds an automated cloud-based public health surveillance system using CDC data.

## Goal
To ingest daily case/death data, transform it using AWS ETL services, and visualize trends in a dashboard.

## Tech Stack
- AWS S3 (data lake storage)
- AWS Lambda (automated ingestion)
- AWS Glue (ETL + cleaning)
- Athena (SQL analytics)
- QuickSight (dashboard)

## Data Source
CDC Open Data Portal –VSRR Provisional Maternal Death Counts and Rates

## Pipeline Architecture
Raw → Clean → Curated → Dashboard
